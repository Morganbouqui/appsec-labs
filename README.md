# 🛡️ AppSec Labs

Welcome to my Application Security Labs.

This repository documents my hands-on journey to becoming an Application Security Engineer through practical PHP projects, secure coding exercises, and OWASP Top 10 vulnerability labs.

Each lab contains:

- Vulnerable implementation
- Secure implementation
- Attack demonstration
- Remediation
- Lessons learned
- Screenshots where applicable

The goal of this repository is not just to learn security concepts, but to understand how to identify, exploit, and fix real-world web application vulnerabilities.
